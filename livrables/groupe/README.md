# Livrables de groupe

Ce dossier contient uniquement les elements destines au rendu commun et a la reproduction du projet.

## Structure

- `01_rapport/`: source LaTeX du rapport et PDF compile
- `02_captures/`: captures retenues pour le rapport ou la soutenance
- `03_csv_exemples/`: CSV propres et representatifs
- `04_annexes/`: extraits commentes, traces de validation et fonctions utiles

Tout fichier ajoute ici doit etre:

- comprehensible par le groupe
- exploitable pour le rendu
- nettoye
- non sensible
